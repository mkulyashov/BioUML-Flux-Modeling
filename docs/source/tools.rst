Перечень используемых инструментов
==================================

В данном разделе приведён список отдельных инструментов, используемых
в конвейере реконструкции потоковых моделей. Каждый инструмент реализован
как WDL-workflow и запускается внутри Docker-контейнера.

Общий обзор инструментов
------------------------

Инструменты для сборки и аннотации бактериального генома
-----------------------------------------------

Ниже приведены используемые инструменты с указанием версии программы
(по данным Docker-образов StaPH-B на Docker Hub):

* fastp, версия 1.0.1  (образ ``staphb/fastp:latest``)
* FastQC, версия 0.11.9  (образ ``staphb/fastqc:latest``)
* SPAdes, версия 3.15.5  (образ ``staphb/spades:3.15.5``)
* QUAST, версия 5.0.2  (образ ``staphb/quast:latest``)
* Prokka, версия 1.14.5  (образ ``staphb/prokka:latest``)

Инструменты для реконструкции и анализа моделей
-----------------------------------------------

Ниже перечислены инструменты (по названию и версии), используемые в WDL-сценариях:

* ModelSEED, версия 1.0.0 (образ ``developmentontheedge/modelseed:1.0.0``)
* gapseq, тег ``latest`` (образ ``cdiener/gapseq:latest``; на Docker Hub также доступны версионированные теги, например ``2024.08.2``)
* Bactabolize, версия 1.0.0 (образ ``developmentontheedge/bactabolize:1.0.0``)
* CobraMod, версия 1.0.0 (образ ``developmentontheedge/cobramod:1.0.0``)
* memote, тег ``latest`` (образ ``developmentontheedge/memote:latest``)
* BioEMMA, версия 1.0.0 (образ ``developmentontheedge/bioemma:1.0.0``)

Подробное описание параметров, входных/выходных файлов и примеров запуска
каждого инструмента приведено в разделе :doc:`individual_tools`.
